MyDraw - A Simple 2D Drawing Program

About

MyDraw is a simple 2D drawing program using basic GL and GLUT functionality. We (Anirudh Vemula and Rohan Prinja) wrote MyDraw for the first homework assignment of our Computer Graphics course.
